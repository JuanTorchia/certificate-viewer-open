## Plugin Name
X.509 Certificate Viewer

## Short Description
A premium tool for inspecting X.509 certificates (PEM, DER, CRT, CER) and Java Keystores (JKS, JCEKS, PKCS12) with a beautiful, expert-level UI.

## Full Description
Expert-grade X.509 certificate inspection tool designed for developers. Stop using command-line tools or external websites to check your certificates. View everything directly inside your IDE with a premium, card-based interface.

### Key Features:
- **Comprehensive Format Support**: Full support for standard certificates (.pem, .crt, .cer) and Keystores (.p12, .pfx, .jks, .jceks).
- **Beautiful Detail View**: v2.0 UI featuring visual status badges (VALID, EXPIRED, SELF-SIGNED).
- **Validity Tracking**: Visual progress bars showing exactly how much time is left until expiration.
- **Expert Metadata**: Display Serial Numbers, Issuer/Subject X.500 names, and SHA-256 fingerprints.
- **Enhanced Productivity**: One-click "Copy" actions for all metadata fields.
- **Seamless Integration**: Opens as a native Editor tab when double-clicking files.
- **Secure**: Interactive password handling for encrypted files.
